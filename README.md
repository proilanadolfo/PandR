# P&R Tea House

A modern, responsive landing page concept for **P&R Tea House Main San Fernando**. The site is built with plain HTML, CSS, and JavaScript and is designed to feel clean, premium, and ready for presentation.

## Overview

This project showcases:
- a polished hero section with the actual storefront image
- a categorized menu for drinks, desserts, snacks, and food meals
- a customer gallery using real photos
- an about section, contact details, and opening hours
- smooth scrolling, animations, and a responsive mobile menu

## Features

- Modern white-and-blue cafГ© theme
- Responsive layout for desktop and mobile
- Sticky navigation with animated dropdown on mobile
- Hero section using [img/Home.jpg](img/Home.jpg)
- Interactive menu filter buttons
- Gallery section using customer photos from `img/Gallery/`
- Clean contact section with the business address
- Smooth hover effects and scroll animations

## How to Use

1. Open [index.html](index.html) in a browser.
2. Make sure the `img/` folder keeps the same file names.
3. Update text, prices, or photos any time by editing the HTML.

## Notes

- If an image is missing, the layout may still work but the photo area will not display correctly.
- The project is static and does not require a backend.
- The current setup is intended for presentation and sample business use.

The final page is a modern, clean, and business-ready sample website for P&R Tea House.
